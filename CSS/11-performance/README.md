# Module 11 — Performance

## Overview

CSS performance is about understanding what's expensive and what's cheap. This module covers layout thrashing, repaint costs, selector performance, font loading, critical CSS, and how to profile with DevTools.

## Prerequisites

- Module 10: Painting & Compositing (paint tiers, compositor layers)
- Module 01: Browser Rendering (rendering pipeline)

## Lessons

| # | Lesson | Topics |
|---|--------|--------|
| 01 | [Layout Thrashing & Reflow](01-layout-thrashing.md) | Forced synchronous layout, read/write batching, ResizeObserver |
| 02 | [Selector & Style Performance](02-selector-performance.md) | Selector matching, specificity overhead, style recalculation scope |
| 03 | [Critical CSS & Loading](03-critical-css.md) | Render-blocking CSS, critical/deferred CSS, font loading, preload |
| 04 | [Profiling & Measurement](04-profiling.md) | Performance panel, Coverage tool, Lighthouse, Core Web Vitals, CLS |
