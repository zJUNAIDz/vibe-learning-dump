
# 01-appendices-further-reading.md

- **Purpose**: A curated list of high-quality resources for continuing your Git education.

---

### Official and Foundational Resources

1.  **Pro Git Book (2nd Edition)**
    - **Link**: [https://git-scm.com/book/en/v2](https://git-scm.com/book/en/v2)
    - **Author**: Scott Chacon and Ben Straub
    - **Why it's great**: This is the official, definitive guide to Git, available for free online. It covers everything from the absolute basics to the deep internals. The chapters on "Git Internals" are particularly relevant to the philosophy of this course. If you read only one book, make it this one.

2.  **The Official Git Documentation**
    - **Link**: [https://git-scm.com/doc](https://git-scm.com/doc)
    - **Why it's great**: The `man` pages for Git are incredibly detailed. When you need to know exactly what a specific command flag does, the official documentation is the source of truth. Learning to read and navigate these pages is a valuable skill.

### Interactive Learning and Visualization

1.  **Learn Git Branching**
    - **Link**: [https://learngitbranching.js.org/](https://learngitbranching.js.org/)
    - **Why it's great**: An interactive, in-browser tutorial that visualizes the commit graph as you perform commands. It's arguably the best way to build an intuitive mental model of how commands like `rebase`, `cherry-pick`, and `reset` affect the graph.

2.  **GitKraken's Git Educational Resources**
    - **Link**: [https://www.gitkraken.com/learn-git](https://www.gitkraken.com/learn-git)
    - **Why it's great**: A collection of high-quality tutorials and videos on a wide range of Git topics, from beginner to advanced. GitKraken is a GUI client, but their educational materials are excellent for all Git users.

### Advanced Topics and Articles

1.  **"A successful Git branching model" by Vincent Driessen**
    - **Link**: [https://nvie.com/posts/a-successful-git-branching-model/](https://nvie.com/posts/a-successful-git-branching-model/)
    - **Why it's great**: The original blog post that introduced the world to GitFlow. It's essential reading for understanding the philosophy behind this highly influential branching strategy.

2.  **GitHub's Guide to GitHub Flow**
    - **Link**: [https://docs.github.com/en/get-started/quickstart/github-flow](https://docs.github.com/en/get-started/quickstart/github-flow)
    - **Why it's great**: A simple, clear explanation of the lightweight branching model used by GitHub and many other web companies.

3.  **Trunk-Based Development Website**
    - **Link**: [https://trunkbaseddevelopment.com/](https://trunkbaseddevelopment.com/)
    - **Why it's great**: A comprehensive resource on the principles, practices, and prerequisites for Trunk-Based Development.

4.  **"Oh Shit, Git!?! Git Commands for Mistakes"**
    - **Link**: [https://ohshitgit.com/](https://ohshitgit.com/)
    - **Why it's great**: A humorous but extremely practical guide to getting yourself out of common Git messes. It's a great resource for learning recovery techniques.

5.  **The Git-SCM YouTube Channel**
    - **Link**: [https://www.youtube.com/c/Git-SCM](https://www.youtube.com/c/Git-SCM)
    - **Why it's great**: The official YouTube channel for the Git project. It includes deep-dive talks from Git contributors and maintainers on advanced topics like performance, new features, and internals.

### Tooling and Productivity

1.  **Conventional Commits Specification**
    - **Link**: [https://www.conventionalcommits.org/](https://www.conventionalcommits.org/)
    - **Why it's great**: The official specification for the commit message format discussed in this course. It's a valuable standard for teams to adopt.

2.  **`git-filter-repo` Documentation**
    - **Link**: [https://github.com/newren/git-filter-repo/blob/main/Documentation/git-filter-repo.txt](https://github.com/newren/git-filter-repo/blob/main/Documentation/git-filter-repo.txt)
    - **Why it's great**: The official documentation for the modern history-rewriting tool. It's dense but comprehensive.

3.  **Husky Documentation**
    - **Link**: [https://typicode.github.io/husky/](https://typicode.github.io/husky/)
    - **Why it's great**: For teams in the JavaScript ecosystem, this is the go-to tool for managing Git hooks.

This list provides a solid foundation for a lifetime of learning about Git. The key is to stay curious, experiment in safe repositories, and always strive to understand the "why" behind the commands.
