# Module 05: Smart Contracts Fundamentals

**Focus:** What smart contracts actually are (and aren't).

## What You'll Learn

The term "smart contract" is misleading. They're neither smart nor contracts. This module demystifies what they really are: **deterministic programs that execute on-chain**.
