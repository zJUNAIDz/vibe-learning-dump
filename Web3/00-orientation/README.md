# Module 00: Orientation

## What Web3 Actually Is (And Isn't)

---

## Overview

Before learning how blockchains work, you need to understand:
- What problem Web3 tries to solve
- What problems it creates
- When Web3 makes sense
- When Web3 is a terrible idea

This module is mandatory. Do not skip it.

## Learning Objectives

By the end of this module, you will:
- ✅ Understand what problem blockchains solve
- ✅ Recognize when Web3 is appropriate (and when it's not)
- ✅ Explain core tradeoffs without hype
- ✅ Critically evaluate Web3 claims
